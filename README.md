# Go Calculator

Консольный калькулятор с разбором математических выражений: приоритет операторов, скобки, переменные, функции, история вычислений.

## Возможности

```
> 3 + 4 * 2
11
> (3 + 4) * 2
14
> 2 ^ 10
1024
> sqrt(16) + abs(-5)
9
> x = 5
5
> x * 2 + 1
11
> ans + 10
21
```

- Операторы `+ - * / % ^` с правильным приоритетом
- Скобки
- Переменные и `ans` (результат последнего вычисления)
- Функции: `sqrt`, `abs`, `sin`, `cos`, `tan`, `log`, `log10`, `floor`, `ceil`, `round`, `pow`, `max`, `min`
- История вычислений (`history`)
- Режим одноразового вычисления: `calc --expr "2+2*2"`

## Структура проекта

```
cmd/calc            точка входа
internal/lexer       токенизация строки
internal/parser       парсер выражений с приоритетами операторов
internal/ast          узлы дерева разбора
internal/evaluator     вычисление, функции, переменные
internal/repl          интерактивный цикл, история, команды
```

Выражение проходит путь: строка → токены (lexer) → дерево разбора (parser) → результат (evaluator).

## Запуск

```bash
go run ./cmd/calc
```

Через Docker:
```bash
docker build -t go-calculator .
docker run -it --rm go-calculator
```

Одноразовое вычисление:
```bash
go run ./cmd/calc --expr "sqrt(144) + 2^3"
```
